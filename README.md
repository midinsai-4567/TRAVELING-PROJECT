Smart Travelling & Tourism Recommendation System using ml

This project is a Smart Travelling & Tourism Recommendation System developed using Python and Flask.
The system recommends tourist places and hotels based on the user budget.

Features
Budget-based travel recommendations
Tourist place suggestions
Hotel recommendations
User data storage using SQLite
Admin panel to view users
Image support for tourist places
Technologies Used
Python
Flask
SQLite
Pandas
HTML/CSS
Files
app.py → Main application
dataset.py → Recommendation logic
config.py → Database configuration
download_images.py → Image downloader
requirements.txt → Required libraries
Installation

Install required packages:

pip install -r requirements.txt

Run the project:

python app.py

Open in browser:

http://127.0.0.1:5000
